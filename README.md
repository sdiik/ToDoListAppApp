## ToDoListAppApp

A simple and elegant To-Do List application built with SwiftUI and the MVVM architecture. This project was created as a testing ground to demonstrate the use of modern UI development in iOS, showcasing clean code practices and efficient state management.

## Features:

Add, edit, delete and detail tasks.
Responsive and adaptive design for iPhone
Follows MVVM principles for separation of concerns and scalability.
Utilizes SwiftUI for declarative UI and smooth animations.
